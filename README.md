# CS425Proj2
Python Flask database-backed web app for a school library.

## Requirements:
- Python 3.4 or later
- Python modules: flask, flask-mysql
- MySQL 5.6 or later (use 'root' as username, and 'pass' as password)

## Run:
Run app.py and go to the url given.
